# 나만의 웹사이트 제작

## 프로젝트 목표
JavaScript를 이용해 나만의 웹 사이트 제작 및 JavaScript 연습

## 프로젝트에서 사용한 기술

JavaScript ES6
라이브러리를 사용하지 않고 바닐라 JS로 작성

## 기능

### 시계
* Datd 객체를 이용해 현재 시간 표시

### to-do list
* 일정 추가&삭제 기능
* 로컬 스토리지를 이용해 일정 저장가능

### 실시간 날씨
* OpenWeather API를 이용해 사용자의 위치정보를  수집하여 실시간 날씨 출력 가능

### 반응형 웹 페이지

* pc,모바일 등 해상도에 따라 보이도록 미디어 쿼리
